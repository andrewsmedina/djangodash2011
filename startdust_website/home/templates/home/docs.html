Installing

The easiest way to install stardust client, is via pip, the python package manager, as so:

pip install stardust

After that add startdust middlware in your settings.py:

MIDDLEWARE_CLASSES = ( 'django.middleware.common.CommonMiddleware',
'django.contrib.sessions.middleware.SessionMiddleware',
...
'stardust_client.middleware.StartDustMiddleware',
)

Add settings for your account and project. (Before that you need be a project registered in our website):

STARDUST_USERNAME = 'username'
STARDUST_PASSWORD = 'password'
STARDUST_PROJECT_TOKEN = 'project-token'
